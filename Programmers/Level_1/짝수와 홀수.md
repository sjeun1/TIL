### 짝수와 홀수

#### 문제 설명
정수 num이 짝수일 경우 "Even"을 반환하고 홀수인 경우 "Odd"를 반환하는 함수, solution을 완성해주세요.<br>
<br>

#### 제한 조건
num은 int 범위의 정수입니다.<br>
0은 짝수입니다.<br>
<br>
<br>
#### 입출력 예<br>
num	return<br>
3	"Odd"<br>
4	"Even"<br>


```
class Solution {
    public String solution(int num) {
        return (num % 2 == 0) ? "Even" : "Odd";
    }
}
```
