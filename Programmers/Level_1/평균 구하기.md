### 평균 구하기

#### 문제 설명
정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성해보세요.<br>
<br>

#### 제한 조건
arr은 길이 1 이상, 100 이하인 배열입니다.<br>
arr의 원소는 -10,000 이상 10,000 이하인 정수입니다.<br>
<br>
<br>
입출력 예<br>
arr	      return<br>
[1,2,3,4]	 2.5<br>
[5,5]	     5<br>


```
class Solution {
    public double solution(int[] arr) {
        double answer = 0;
        int length = arr.length;
        
        for(int num : arr) {
            answer += num;
        }
        return answer / length ;
    }
}
```
