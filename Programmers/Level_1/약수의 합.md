### 약수의 합

#### 문제 설명
정수 n을 입력받아 n의 약수를 모두 더한 값을 리턴하는 함수, solution을 완성해주세요.<br>
<br>

#### 제한 조건
n은 0 이상 3000이하인 정수입니다.<br>
<br>
<br>
#### 입출력 예<br>
n	  return<br>
12	28<br>
5	  6<br>

#### 입출력 예 설명<br>
##### 입출력 예 #1<br>
12의 약수는 1, 2, 3, 4, 6, 12입니다. 이를 모두 더하면 28입니다.<br>

##### 입출력 예 #2<br>
5의 약수는 1, 5입니다. 이를 모두 더하면 6입니다.<br>

```
class Solution {
    public int solution(int n) {
        int answer = 0;
        for(int i=1; i<=n; i++) {
            if(n%i == 0) answer += i;
        }
        return answer;
    }
}
```
